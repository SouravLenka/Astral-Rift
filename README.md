# Astra: Rift

A modular 2D local co-op roguelite space shooter built in Unity 6 LTS.

## Project Overview

Astra: Rift is designed as a production-quality game with an extensible architecture, data-driven systems, and polished gameplay.

## Development Roadmap

- Milestone 1: Project foundation, folder structure, version control, bootstrap scene.
- Milestone 2: Player systems and input.
- Milestone 3: Weapon and combat systems.
- Milestone 4: Enemy systems.
- Milestone 5: Wave system.
- Milestone 6: Power-ups.
- Milestone 7: Upgrade system.
- Milestone 8: Boss system.
- Milestone 9: UI.
- Milestone 10: Audio.
- Milestone 11: Visual polish.
- Milestone 12: Save system.
- Milestone 13: Hangar.
- Milestone 14: Environments.
- Milestone 15: Optimization and QA.

  - Resources/
  - StreamingAssets/
  - Settings/

## Coding Standards

- Use meaningful names.
- Prefer serialized private fields.
- Use ScriptableObjects for game data.
- Keep systems modular and event-driven.
- Avoid hardcoded values.
- Cache references and minimize allocations.
